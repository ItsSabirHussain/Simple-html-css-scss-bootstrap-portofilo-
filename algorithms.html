Algorithms

In mathematical and computational point of view, the algorithm is well defined set of instruction that can be implemented by computer to solve some sort problem by computation [1][2]. Every algorithm has finate time and space complexity to ensure effectiveness [3].For writing algorithms, there must be a well-defined and simple language used[4]. Every algorithm has an initial state, some sequence of instruction that must be followed to reach at termination state to solve computational problem [5].
There are many algorithmic techniques that are used to design algorithm’s including brute force approach, greedy algorithms, divide and conquer, branch and bound, backtracking and dynamic programming.
Brute force approach, it is very naïve problem-solving method to trying all possible solution to check if there is any way to solve a problem [6]. Greedy is problem solving technique in which the most optimal choice is chose from available choices in other words local optimal is point of concern [7]. Divide and conquer is another problem-solving approach in which problem is divided into smaller sub-problems to solve them easily [8]. Branch and bound is an algorithmic technique in which some condition are check on each possible solution but using breath first search all branches those are not satisfying condition will eliminated [9] and opposite to it backtracking using same paradigm but using depth first search and backtrack if condition is not satisfied [10]. Dynamic programming is another problem-solving technique that is used to find global optimal solution and save time to compute overlapping subproblem by storing results [11].
